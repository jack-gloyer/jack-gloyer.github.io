Best Practices when using Classes vs IDs

Major Differences in the HTML attributes "Classes" and "ids"

This is a short post what classes and ids are and when to best use each attribute.

Classes

Classes are attributes which can be reused on multiple HTML elements.

They are declared with a "." at the start of the declaration.

These can be declared either in the head section of an HTML file or on a CSS stylesheet.

You can use an unlimited amount of styling in a Class.

You can also use multiple classes on the same element.

IDs

IDs are unique attributes. They cannot be reused on multiple HTML elements.

IDs are declared with a "#" at the start of the declaration.

They are also declared in either the header of an HTML file or on a CSS stylesheet.

Only one ID element can be used on a page.

Unlike classes, only one ID can be used in an element.

They are also useful for jumping between places on the same web page.
